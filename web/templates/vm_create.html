Источники VM

Открыть диски
ISO-установщик Классический сценарий: создаём пустой qcow2-диск и устанавливаем систему с ISO.
Готовый диск Для Raspberry/Orange Pi образов и готовых .img/.raw/.qcow2. Virtuality импортирует образ в отдельный диск VM.
Сеть Для VPS выбирай NAT Router. Bridge нужен только если br0 реально есть на сервере. 
ISO:    virt-install --cdrom ubuntu.iso --disk VM.qcow2
IMG:    qemu-img convert -O qcow2 image.img VM.qcow2
        virt-install --import --disk VM.qcow2
