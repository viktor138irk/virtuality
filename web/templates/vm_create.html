Сетевые режимы

Открыть сеть
VPS NAT Router Правильный режим для VPS с одним внешним IP. VM сидят за NAT, входящие порты пробрасываются вручную.
Bridge br0 Режим для домашнего или офисного сервера, где VM могут получать IP из локальной сети. 
VPS:    --network network=virtuality-nat,model=virtio
LAN:    --network bridge=br0,model=virtio
