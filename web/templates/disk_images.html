Файл попадёт в /var/lib/virtuality/disk-images. При создании VM образ будет скопирован/сконвертирован в отдельный qcow2-диск VM.

Как использовать

import disk
Raspberry / Orange Pi images Загрузи .img, затем в “Создать VM” выбери режим “Готовый диск”.
Оригинал не трогаем Virtuality делает отдельный диск VM, чтобы загруженный образ оставался шаблоном. 
qemu-img convert -O qcow2 source.img VM.qcow2
virt-install --import --disk path=VM.qcow2
